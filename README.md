# vue3-cookbook

用来熟悉 vue3.x，及其生态的实验项目



## Install

```bash
# vue3 旧版
npm install vue@next

# vue3 新版
npm init vue@latest

# @vue/cli 5.0.8
npm install -g @vue/cli
npm update -g @vue/cli
vue create vue-cli-demo

Vue CLI v5.0.8
```
